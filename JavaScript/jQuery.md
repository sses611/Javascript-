jQuery 사용법
- CDN
http://jquery.com/download/에서 파일 다운로드 → compressed 주로 사용
lib폴더에 아래코드 연결

```
<script src="/lib/jquery-1.11.0.min.js"></script>
```

jQuery를 이용한 노트 웹요소 찾기
const $변수이름 = $("CSS선택자") : 가장 기초적인 노드 찾는 방법
